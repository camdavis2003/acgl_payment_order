Financial Management System (FMS)

The concept for the ACGL Financial Management System began during the return flight from the 2026 Conference of Grand Masters and Grand Secretaries of North America in Chicago, Illinois. As Most Worshipful Grand Master Arda Çilingir and Right Worshipful Grand Secretary Cameron J. Davis discussed the administrative tasks awaiting them upon returning home, attention turned to the process of submitting travel reimbursement requests from the conference.

The Grand Master remarked that it would be helpful if the Grand Lodge had a way to submit these requests electronically rather than relying on traditional manual methods. That simple observation sparked an idea.

During the flight back to Frankfurt, Germany, the Grand Secretary began writing the foundational programming code for what would become the ACGL Financial Management System (FMS). By the time the aircraft landed, the initial framework of the system had already been developed, marking the beginning of a platform that now supports the Grand Lodge's financial workflow.

This application is a lightweight, browser-based financial workflow tool for managing Payment Order requests and year-based budget activity. It supports a complete request-to-record process with shared storage, role-based access, and consistent navigation across modules.

Modules include: New Request Form, Budget (with Dashboard), Income, Payment Orders (including Reconciliation), Ledger, and Settings. Data can run in standalone mode (browser storage) or in WordPress shared mode.

Backups: create a WordPress backup for the active year, restore from a backup file, and (when enabled) manage Cloud Backups in Google Drive.
