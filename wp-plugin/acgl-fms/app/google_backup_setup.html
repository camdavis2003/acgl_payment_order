Google Backup Setup

Follow these steps once in WordPress admin to enable Cloud Backup and restore from Google Drive.

  1. In WordPress, open ACGL FMS settings and find the Google Drive Backup configuration.
  2. Enter your Google OAuth Client ID and Client Secret for your project.
  3. Save settings, then click the Google authorization button and complete consent.
  4. Return to the Backup section in ACGL FMS and click Cloud Backup to upload.
  5. Use Download or Restore on a cloud backup entry when needed.

Tip: Cloud backup stores one all-years file. WordPress backup cards remain year-based.
